# Описание Лабораторной работы №7

**Flexbox** (Flexible Box Layout) — это модуль CSS для создания гибких одномерных 
раскладок. Он идеально подходит для:
- Выравнивания элементов по горизонтали или вертикали
- Распределения свободного пространства
- Создания навигационных меню
- Центрирования элементов

## 📌 Основные понятия Flexbox

### Flex-контейнер
Родительский элемент с `display: flex`. Управляет расположением дочерних flex-элементов.

### Flex-элементы
Дочерние элементы внутри flex-контейнера.

---

## Эксперименты с Flexbox (index.html)

### 1. `flex-direction`
- `row` (по умолчанию) — элементы в строку.
- `column` — элементы в столбец.
- `row-reverse` — строка справа налево.
- `column-reverse` — столбец снизу вверх.

### 2. `align-items`
- `stretch` — растягивание по высоте.
- `flex-start` — прижаты к верху.
- `flex-end` — прижаты к низу.
- `center` — по центру.
- `baseline` — по базовой линии текста.

### 3. `flex-wrap`
- `nowrap` — без переноса.
- `wrap` — перенос на новую строку.
- `wrap-reverse` — перенос в обратном направлении.

### 4. `gap`
Задание отступов между flex-элементами (вместо `margin`).

### 5. Свойства flex-элементов
- `flex-grow` — коэффициент растяжения.
- `flex-basis` — базовый размер.
- `flex-shrink` — коэффициент сжатия.
- `align-self` — индивидуальное выравнивание.

---

## Практические примеры (flexbox.html)

### Пример 1: Навигационное меню
Горизонтальное меню с равномерным распределением ссылок.

```css
.navbar {
    display: flex;
    justify-content: space-around;
    background-color: #333;
    padding: 15px;
}
```

### Пример 2: Карточки товаров
Адаптивные карточки с прижатой кнопкой внизу.
```css
.cards {
    display: flex;
    gap: 20px;
    flex-wrap: wrap;
}

.card {
    flex: 1 1 250px;
    display: flex;
    flex-direction: column;
}

.card button {
    margin-top: auto;
}
```
